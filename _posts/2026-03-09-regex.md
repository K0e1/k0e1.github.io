### 正则表达式

***

用于匹配规律规则的表达式，正则表达式最初是科学家对人类神经系统的工作原理的早期研究，现在在编程语言中有广泛的应用。正则表通常被用来检索、替换那些符合某个模式(规则)的文本。正则表达式是对字符串操作的一种逻辑公式，就是用事先定义好的一些特定字符、及这些特定字符的组合组成一个“规则字符串"，这个“规则字符串"用来表达对字符串的一种过滤逻辑。

#### 正则表达式的作用

***

*   1.给定的字符串是否符合正则表达式的过滤逻辑(匹配)
*   2.可以通过正则表达式，从字符串中获取我们想要的特定部分(提取)
*   3.强大的字符串替换能力(替换)

#### 正则表达式的组成

***

*   普通字符
*   特殊字符(元字符):正则表达式中有特殊意义的字符

***

#### 常用的元字符

| 元字符 | 说明                           |
| ------ | ------------------------------ |
| \d     | 匹配数字                       |
| \D     | 匹配任意非数字字符             |
| \w     | 匹配字母或数字或下划线         |
| \W     | 匹配任意不是字母，数字，下划线 |
| \s     | 匹配任意空白符                 |
| \S     | 匹配任意非空白符               |
| .      | 匹配除空白符之外的任意字符     |
| ^      | 表示匹配行首的文本(以谁开始)   |
| \$     | 表示匹配行末的文本(以谁结束)   |

*   加开始和结束是精确匹配

***

#### 限定符

| 限定符 | 说明                        |
| ------ | --------------------------- |

| +      | 修饰前一个字符，重复1或多次 |
| ?      | 修饰前一个字符，重复0或1次  |

| {n,}   | 修饰前一个字符，重复n或多次 |

#### 特殊字符

***

1.  \[]字符串用中括号括起来，表示匹配其中的任一字符，相当于或的意思
2.  \[^]匹配除中括号以内的内容
3.  \转义符
4.  │或者，选择两者中的一个。注意|将左右两边分为两部分，而不管左右两边有多长多乱
5.  ()从两个直接量中选择一个，分组eg\:gr(a|e)y匹配gray和grey
6.  \[\u4eoo-\u9fa5]匹配汉字
7.  /num 引用第num组的内容
8.  （第一个是第一组，全部正则表达式是第0组，第二个左括号（是第二组
    1.  (?P<分组名>)    设置分组	 (?P<分组名>)使用分组

#### python中正则表达式常见函数

| 函数                                               | 功能描述                                                     | 返回值                                               | 常用参数                                                     |
| :------------------------------------------------- | :----------------------------------------------------------- | :--------------------------------------------------- | :----------------------------------------------------------- |
| `re.match(pattern, string, flags=0)`               | 从字符串的**开头**匹配正则表达式，只匹配一次。               | 匹配成功返回 `Match` 对象，否则返回 `None`。         | `pattern`: 正则表达式 `string`: 待匹配字符串 `flags`: 标志位（如 re.I） |
| `re.search(pattern, string, flags=0)`              | 扫描**整个**字符串，返回第一个成功匹配的位置。               | 匹配成功返回 `Match` 对象，否则返回 `None`。         | 同上                                                         |
| `re.findall(pattern, string, flags=0)`             | 找到字符串中所有**非重叠**匹配，以列表形式返回。             | 如果模式中有分组，返回元组列表；否则返回字符串列表。 | 同上                                                         |
| `re.finditer(pattern, string, flags=0)`            | 找到所有匹配，返回一个迭代器，迭代产生每个匹配的 `Match` 对象。 | 迭代器，每个元素是 `Match` 对象。                    | 同上                                                         |
| `re.sub(pattern, repl, string, count=0, flags=0)`  | 用 `repl` 替换字符串中所有匹配（或最多 `count` 次）的子串。  | 替换后的新字符串。                                   | `repl`: 替换内容（可以是字符串或函数） `count`: 最大替换次数，0表示全部 |
| `re.subn(pattern, repl, string, count=0, flags=0)` | 功能同 `sub`，但返回一个元组 `(新字符串, 替换次数)`。        | 元组 `(new_string, number_of_subs_made)`。           | 同上                                                         |
| `re.split(pattern, string, maxsplit=0, flags=0)`   | 用匹配到的子串作为分隔符，分割字符串。                       | 分割后的字符串列表。                                 | `maxsplit`: 最大分割次数，0表示不限制                        |
| `re.compile(pattern, flags=0)`                     | 编译正则表达式，返回一个 `Pattern` 对象，之后可以调用其方法（如 `match`、`search` 等）进行复用。 | `Pattern` 对象（拥有 `match`、`search` 等方法）。    | 同上                                                         |
